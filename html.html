Josh Introduction

Josh Laravel Admin Template + Front End + CRUD Builder

Josh is a user friendly admin template with GUI CRUD builder. A lot of time can be saved by using CRUD on repetitive tasks. The template is built with HTML, CSS, Bootstrap and Laravel versions, and also comes with authentication by default. The template has every component required for designing a website. Some other awesome features are:

1 Versions - HTML, Laravel
2UI - Bootstrap, Material Version
3CRUD Builder
4Front End Theme
5Laravel Examples
6Blog Module.

Josh Features

Josh Laravel Admin Template has Awesome Features

FRONTEND

Josh Admin Template has attractive and well designed Frontend pages which are completely Responsive.

BACKEND

Backend Applications in the Josh Admin Template are developed by Laravel which is customizable.

MATERIAL VERSION

Josh has material version for its backend with other features like ripples and animated styles.

LARAVEL

Josh is built using laravel and also comes with authentication by default, also includes many laravel examples.

CRUD

By using CRUD, data can be easily added and deleted. It saves lot of time on repetitive tasks.

BOOTSTRAP

Josh uses SaSS for customized look and responsiveness and it is built with Bootstrap.

SPARK SKIN

When a new Laravel app is created, functionality can be recreated by using Laravel Spark Skin.

USER MANAGEMENT

The features like Login, Edit and Register can be accessed through this section.

ADVANCED DATATABLES

Advanced Datatables have great features of toggling, rearranging data that makes data analysis easy.

CUSTOM DATATABLES

Data is retrieved from backend by using AJAX and filters are provided for easy access and analysis of data

LOG VIEWER

All the Laravel errors raised in the project can be shown here, so that errors can be identified easily.

ACTIVITY LOG

User Activity in the website can be logged through Activity Log such as Login, Edit and Logout.

LARAVEL CHARTS

Dynamic Charts examples are added which utilizes data in the database for quick analysis of data.

SELECT2 FILTER

The suggestions in the dropdown shows only related data in order to reduce burden on the database.

MINI SIDEBAR

Here comes Minisidebar. However, sidebar can be used as both normal sidebar and mini sidebar.

Josh Starter Kit
Josh Laravel Admin Template has Starterkit

Josh Admin Template comes with more than 100 pages. For an instance, if the project requires only few pages to add in the application we came up with a concept called Starter Kit where we include minimum required pages like Login, Register, Forgot Password and User Management etc., Besides this if the developer requires any additional pages, they can get it from main version of Josh, we are providing detailed documentation of how to include them into starter kit.
